# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in NEXORA, please report it privately by emailing **vishwajeetsrk@gmail.com**.

**Do not** open a public issue for security vulnerabilities.

We will:
1. Acknowledge receipt within 48 hours
2. Provide an initial assessment within 5 business days
3. Work on a fix and release timeline
4. Notify you when the fix is deployed

## Scope

- Authentication flows (sign-in, sign-up, password reset)
- API endpoint security
- Data storage and transmission
- Third-party dependency vulnerabilities

## Responsible Disclosure

We ask that you:
- Give us reasonable time to fix the issue before disclosure
- Do not exploit the vulnerability
- Do not access or modify user data without permission
